Recipe Cook and Rating Prediction Model

November 2021
University of California San Diego - Course Project

This project was implemented as a part of the course Recommender Systems and Web Mining. Using the data from Food.com, the first task was to predict whether a user would cook a recipe. Since data only consisted of entries where user had cooked a recipe and given it some rating, for this task I performed negative sampling from the given dataset. Subsequently, I split the data into train and test set and using feature ablation studies I fit a logistic regression model on the test data using inbuilt Python library. The accuracy achieved for this task was ~70%.
The second task was to predict the rating a user would give a recipe. For this task, I fit a latent factor model on the given dataset yielding a Mean Squared Error ~0.8.
The solutions to both the tasks had to be uploaded on Kaggle (Username: Juzer Golwala)

Home Butler

Dec 2013 - May 2014
A Voice Controlled Robot for the differently abled

I collaborated on the design of an autonomous robot which fetches objects upon a voice or text command. The objects and their locations are pre-fed into a library. So on the command the robot moves towards the object's location and on reaching it an on board camera starts taking images of the location, simultaneously comparing it with already stored images of the object. Upon successful comparison the robot activates a mechanical gripper which then extends towards the object and fetches it. Once the object is gripped it traces the path back to the user along with the object.

LabVIEW, MATLAB, Android and Arduino
