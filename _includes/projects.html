XQuery Processor

January 2022 - March 2022
University of California San Diego - Course Project

This project was a part of the course Database System Implementation. We built an XQuery processor in Java using the ANTLR4 generated classes to parse the query and read the XML data using DOM. Subsequently, we developed an XQuery evaluator that would return the results by visiting the parsed nodes in the abstract syntax tree of the query. Lastly, if the input query conformed to a subset of XQuery grammar, it would undergo re-writing and be executed using join optimizations.

Java, XQuery, XMl, ANTLR4

Recipe Cook and Rating Prediction Model

November 2021
University of California San Diego - Course Project

This project was a part of the course Recommender Systems and Web Mining. Using the data from Food.com, the first task was to predict whether a user would cook a recipe. The dataset only consisted of entries where a user had cooked a recipe and given it some rating. Hence, I had to perform negative sampling from the given dataset. Subsequently, I split the data into train and test sets and used feature ablation studies to fit a logistic regression model on the train data using inbuilt Python library. The accuracy achieved for this task was ~70%.
The second task was to predict the rating a user would give a recipe. For this task, I fit a latent factor model on the given dataset yielding a Mean Squared Error of ~0.8.
The solutions to both the tasks are on Kaggle. (Username: Juzer Golwala).

Python, Data Analysis, Machine Learning

Advancing Recommender Systems by Mitigating

Shilling Attacks

July 2017 - May 2018
Sardar Patel Institute of Technology - Final Year Project

A year-long project for which we reviewed existing research in the area of Recommender Systems and various difficulties in building smooth functioning recommender engines. We shortlisted shilling attacks as our research topic. We studied various kinds of shilling attacks and their effects on different recommender systems. Subsequently, we formulated a statistical model to detect shilling profiles in recommender systems using collaborative filtering (user-user similarity). Our model improved the accuracy of detection of shilling profiles by ~7% (as of existing research).

Python, Data Analysis and Visualization, Machine Learning
